## Installation

```bash
$ npm install
```

```bash
TOKEN = <YOUR_GIT_TOKEN>
OWNER = <REPOSITORY_OWNER>
REPO  = <REPOSITORY_NAME>
```
## Running the app

```bash
# development
$ npm run build
$ npx serverless offline

# test mode
$ jest

# deploy
$ npm run build
$ npx serverless deploy
```
